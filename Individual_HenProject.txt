# 🧩 🐔 Asta-type Poultry System: Main Parts Breakdown

Think of the whole system as **6 functional blocks**, not random machines.

---

# 1. 🫧 Scalding System (Pre-processing)

👉 Purpose: loosen feathers before plucking

### Includes:

* Scalding tank (hot water heater system)
* Temperature controller
* Water circulation/heating unit
* Basket or hanger system for dipping birds

💡 This is the “prep stage”

---

# 2. 🪶 Defeathering System (Plucking Unit)

👉 Purpose: remove feathers mechanically

### Includes:

* Rotary plucking machine (rubber finger drum)
* Water spray system (helps feather removal)
* Motor + gearbox system
* Drain outlet for feathers

💡 This is one of the most important machines

---

# 3. 🧼 Washing System

👉 Purpose: clean carcass after plucking

### Includes:

* Spray washing unit (water jets)
* Stainless steel rinse station
* Drainage tray system
* Optional disinfectant spray system

💡 Hygiene control stage

---

# 4. 🔪 Cutting / Processing System

👉 Purpose: convert whole bird into pieces

### Includes:

* Cutting table (SS steel)
* Manual cutting tools OR motor-assisted cutter
* Portioning guide system
* Safety guards

💡 This is semi-manual in your scale (10 birds/hour)

---

# 5. ♻️ Waste Handling System

👉 Purpose: separate and collect waste safely

### Includes:

* Feather collection chute
* Blood/drain water channel
* Internal waste bin (organs etc.)
* Wash water filtration outlet

💡 VERY important for hygiene

---

# 6. ⚡ Power + Control System

👉 Purpose: run everything safely

### Includes:

* Electric motor supply (single/three phase)
* Control switches / relay box
* Optional Arduino/PLC controller (advanced)
* Safety cutoff system (emergency stop)

💡 This is the “brain + power”

---

# 🧠 2. Optional (advanced modules)

If you upgrade system quality:

* ❄️ Chilling unit (cold water / ice bath)
* 📦 Packaging unit
* 📊 Weight measurement system
* 🧯 Sterilization / cleaning spray system

---

# 🧩 3. Full system structure (simple map)

```id="poultry_flow"
Scalding → Plucking → Washing → Cutting → Output
    ↓         ↓         ↓         ↓
  Heater   Feathers   Drainage   Waste
```

👉 Everything is connected as a **flow system**

---

Asta-type suppliers don’t sell “one machine system”

They sell:
